# AI Marketing Lab OS

The live dashboard for an AI marketing operating system that runs a real agency: six specialist
agents on one brain, hard governance gates, and honest run logs. Every send stops at one human
approval.

This repo hosts the **public dashboard** (`index.html` + `state.json`) served via GitHub Pages. The
engine (agent skills, business context, drafts, client data) runs privately and is never published.

- Live: https://os.aimarketinglabs.in
- State is refreshed by the OS via its `export-state.sh` (sanitized: counts and statuses only).
